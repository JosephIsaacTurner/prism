# API Reference

This section provides a detailed reference for Prism's API.

Prism's core functionality is centered around several key modules:

- **Datasets**: Handled via the `Dataset` class, which manages data loading and analysis configuration.
- **Analysis**: Main statistical engines like `permutation_analysis` and `permutation_analysis_nifti`.
- **Spatial Similarity**: Tools for comparing statistical maps across datasets.
- **Stats**: Underlying statistical functions for GLM modeling.
- **Preprocessing**: Internal utilities for data loading and masking.
